# Shofereat — Brand Identity System

A complete, self-contained brand guidelines site (logo usage, color, typography,
iconography, UI system, mobile app + website mockups, and motion).

Everything is embedded in a single `index.html` — no build step, no dependencies.

## Deploy on Vercel

1. Push this folder to a GitHub repository.
2. Go to https://vercel.com → **Add New → Project → Import** your repo.
3. Framework Preset: **Other**. Build Command: *(leave empty)*. Output Directory: *(leave empty)*.
4. Click **Deploy**. You'll get a live `*.vercel.app` URL.

Or with the Vercel CLI:

```bash
npm i -g vercel
cd shofereat-brand
vercel        # preview deploy
vercel --prod # production deploy
```

## Local preview

Just open `index.html` in any browser.
